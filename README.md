# Zingen

O **Zingen** é uma landing page desenvolvida com **HTML5** e **CSS3** para promover um aplicativo de karaokê. A interface apresenta os principais recursos da plataforma, planos de assinatura e chamadas para download, utilizando um layout moderno, responsivo e organizado em componentes de estilo reutilizáveis.

Este projeto faz parte dos desafios práticos da **Rocketseat**, com foco na criação de interfaces web utilizando apenas HTML e CSS.

---

## ✨ Funcionalidades

* Landing page responsiva.
* Apresentação do aplicativo.
* Seção de funcionalidades.
* Destaque dos benefícios do serviço.
* Planos de assinatura.
* Chamada para download do aplicativo.
* Links para redes sociais.
* Navegação entre seções por âncoras.
* Layout moderno utilizando CSS modular.

## ⚙️ Pré-requisitos

Para executar o projeto é necessário apenas:

* Um navegador moderno (Google Chrome, Firefox, Edge ou Safari).

---

## 🔧 Instalação

Clone o repositório:

```bash
git clone <URL_DO_REPOSITORIO>
```

Acesse a pasta do projeto:

```bash
cd Zingen
```

---

## ▶️ Como executar

Como se trata de um projeto estático, basta abrir o arquivo `index.html` em um navegador.

Durante o desenvolvimento, recomenda-se utilizar a extensão **Live Server** do Visual Studio Code para recarregamento automático da página.
